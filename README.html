How to activate the virtual env and the Django development server?

Note: eCommerce_django is the project folder containing 2 necessary things

  • the virtual env (pyvenv.cfg)
      • activate* the virtual env with .\Scripts\activate when in the same folder as the virtualenv (not in the virtualenv folder itself)
      deactivate* a running virtualenv by typing deactivate into the shell
  • the django/website codebase (yahoo)
      • activate* the website for use on your web browser by running "python manage.py runserver" on the folder containing manage.py (probably the project folder, ie eCommerce_django)
      deactivate* a running development env by entering CTRL+BREAK into the development env in the shell
